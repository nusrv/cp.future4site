# n8n Draft Workflow Exports

These workflow files are sanitized planning/build artifacts for import into n8n after owner review.

Rules:

- `active` must remain `false`.
- No credentials are embedded.
- Placeholder webhook paths must be replaced in n8n.
- Publishing workflows must stay dry-run until Meta permissions, tokens, public asset hosting, duplicate protection, and owner approval are complete.
- Live workflow activation is not performed by this repository.
