# User Administration

## Initial Admin

Use:

```bash
npm run admin:bootstrap
```

The bootstrap script creates an owner administrator only if no active owner admin exists.

## Account Controls

- Add users.
- Edit username, display name, email, role, and status.
- Disable or reactivate users.
- Reset password.
- Force password change.
- Revoke sessions.
- Track last login.

## Deferred

- Two-factor authentication.
- SSO.
- Fine-grained field-level permissions beyond MVP role checks.
