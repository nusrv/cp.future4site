# Design System

The UI is an internal B2B operations interface, not a public marketing page.

## Direction

- Trust-first, dense enough for operational work, still readable.
- Cream base, deep olive navigation, sunflower gold accents.
- Strong information hierarchy with cards, tables, badges, and status panels.
- Avoid decorative visuals that compete with data and review decisions.

## Tokens

- Background: warm cream
- Primary: deep olive
- Accent: sunflower gold
- Status: green for safe/complete, amber for warning/dry-run, red for blocked/error

## Accessibility

- Keep contrast high.
- Use visible focus states.
- Use text labels, not color alone, for status.
- Keep table actions keyboard reachable.
